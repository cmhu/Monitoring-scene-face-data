# Monitoring-scene-face-data
淘然视界红外人脸数据（如有问题，可以随时在Issues中提问）：
1）数据规模：当前165人，持续更新
2）数据种类：图片+框标注
3）采集设备：IPC 
4）种族分布：黄种
5）采集场景：室外、室内
6）年龄分布：未知
7）角度姿态：多角度多姿态
8）人脸表情：多种表情
9）光照分布：多样 
10）数据格式：jpg 
11）应用：人脸识别
